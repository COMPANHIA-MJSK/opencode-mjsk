# ECC - Enhanced Context Control

ECC provides coding rules and best practices for consistent, high-quality code generation.

## Common Rules (all projects)
- Write clean, maintainable, self-documenting code
- Follow SOLID principles and DRY methodology
- Use meaningful variable/function names
- Add error handling for all edge cases
- Write unit tests for critical logic
- Use version control best practices

## TypeScript Rules
- Use strict TypeScript configuration
- Prefer interfaces over types for objects
- Use discriminated unions for state management
- Leverage generics for reusable utilities
- Avoid `any` - use `unknown` and type guards

## Installation
ECC rules are loaded from `~/.config/opencode/skills/ecc/rules/`
