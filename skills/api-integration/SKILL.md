# API Integration & Backend Communication

Expert in building and consuming APIs.

## API Types
- **REST**: Resources, methods, status codes, versioning
- **GraphQL**: Queries, mutations, subscriptions, Apollo
- **WebSockets**: Real-time communication, Socket.io
- **gRPC**: High-performance RPC, protocol buffers
- **Server-Sent Events**: Real-time updates

## Data Formats
- JSON, XML, FormData
- File uploads (multipart)
- Streaming responses
- GraphQL fragments and variables

## Authentication
- JWT (JSON Web Tokens)
- OAuth 2.0 / OIDC
- Session-based auth
- API keys
- Clerk, NextAuth.js, Lucia, Auth0

## Frontend Integration
- **React**: TanStack Query, SWR, Apollo Client, RTK Query
- **State Management**: Server state vs client state
- **Caching**: Cache invalidation, stale-while-revalidate
- **Error Handling**: Error boundaries, retry logic, optimistic updates

## Best Practices
- Proper error handling
- Rate limiting awareness
- Pagination (cursor, offset)
- API versioning
- Request/response validation (Zod)
- Environment-specific endpoints
