# shadcn/ui Development

Expert in building beautiful UIs with shadcn/ui component system.

## Core Concepts
- Copy-paste component architecture
- Tailwind CSS + Radix UI primitives
- Customizable themes and variants
- TypeScript-first components

## Components
- **Layout**: Container, Stack, Grid, Aspect Ratio
- **Forms**: Button, Input, Select, Checkbox, Radio, Switch, Textarea, Form
- **Data Display**: Card, Table, Badge, Avatar, Progress, Skeleton
- **Feedback**: Alert, Dialog, Sheet, Toast, Sonner
- **Navigation**: Tabs, Navigation Menu, Breadcrumb, Dropdown Menu
- **Overlay**: Modal, Popover, Tooltip, Hover Card, Command
- **Charts**: Recharts integration via shadcn charts

## Customization
- Theme variables in globals.css
- Component variants (cva)
- Style overrides and extensions
- Dark mode theming
- Custom component creation

## Best Practices
- Use with Tailwind CSS utilities
- Leverage Radix UI accessibility
- Consistent component API
- Proper form handling with React Hook Form + Zod
