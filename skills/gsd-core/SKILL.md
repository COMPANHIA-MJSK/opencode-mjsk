# GSD Core - Goal Specs Development

GSD Core is a meta-prompting, context engineering, and spec-driven development system.

## Capabilities
- **Meta-Prompting**: Generate optimized prompts for complex tasks
- **Context Engineering**: Structure project context for maximum AI effectiveness
- **Spec-Driven Development**: Create formal specifications before implementation
- **Requirements Analysis**: Break down complex requirements into actionable specs

## Workflow
1. **Analyze** - Understand requirements deeply
2. **Specify** - Create formal specification
3. **Plan** - Break spec into tasks
4. **Implement** - Build incrementally
5. **Verify** - Test against spec

## Commands
- `gsd-core init` - Initialize GSD in a project
- `gsd-core spec` - Create a new specification
- `gsd-core plan` - Generate implementation plan
- `gsd-core review` - Review code against spec
