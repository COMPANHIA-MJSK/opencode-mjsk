# TypeScript Development

Expert in TypeScript type system and best practices.

## Type System
- **Basic Types**: string, number, boolean, array, tuple, enum, union, intersection
- **Advanced**: generics, conditional types, mapped types, template literal types
- **Utility Types**: Partial, Required, Pick, Omit, Record, Exclude, Extract
- **Patterns**: discriminated unions, branded types, type guards, satisfies
- **Configuration**: strict mode, path aliases, declaration files

## Project Setup
- tsconfig.json optimization
- monorepo with TypeScript
- declaration files (*.d.ts)
- module resolution strategies

## Best Practices
- Prefer interfaces for public APIs
- Use type aliases for complex types
- Leverage exhaustiveness checking
- Avoid `any` - use `unknown`
- Use `as const` for literal inference
- Generic constraints with extends
