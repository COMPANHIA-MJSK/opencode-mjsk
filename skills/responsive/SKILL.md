# Responsive Design

Expert in creating responsive, mobile-first web experiences.

## Core Principles
- Mobile-first approach
- Fluid layouts and typography
- Progressive enhancement
- Breakpoint strategy

## Techniques
- **CSS Grid**: Auto-fit, auto-fill, minmax(), grid areas
- **Flexbox**: Wrapping, alignment, ordering
- **Fluid Typography**: clamp(), viewport units, rem/em
- **Responsive Images**: srcset, sizes, picture element, WebP/AVIF
- **Container Queries**: container-type, container-style
- **Media Queries**: width, height, orientation, prefers-reduced-motion, prefers-color-scheme

## Breakpoint Strategy
- Mobile: 320px - 480px
- Tablet: 768px - 1024px
- Desktop: 1280px+
- Large: 1536px+

## Testing
- Chrome DevTools device emulation
- Real device testing
- Responsive design validation
- Cross-browser compatibility

## Frameworks
- Tailwind responsive utilities
- Bootstrap grid
- CSS Grid systems
