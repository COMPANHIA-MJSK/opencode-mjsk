# Animations & Motion Design

Expert in web animations, motion design, and interactive experiences.

## Techniques
- **CSS Animations**: Transitions, keyframes, transforms, animations
- **JavaScript Animations**: GSAP, Framer Motion, Anime.js, Three.js
- **SVG Animations**: Path animations, morphing, stroke animations
- **Canvas/WebGL**: Particle systems, 3D rendering, shaders
- **Scroll Animations**: Scroll-triggered, parallax, reveal effects
- **Micro-interactions**: Hover states, button effects, loading animations
- **UI Motion**: Page transitions, route transitions, shared element animations
- **Performance**: 60fps, will-change, GPU acceleration, RAF

## Best Practices
- Respect `prefers-reduced-motion`
- Use easing functions properly (cubic-bezier)
- Keep animations under 300ms for UI elements
- Use FLIP technique for layout animations
- Lottie for complex vector animations
- Test on low-end devices

## Libraries
- GSAP (GreenSock) - Professional grade
- Framer Motion - React
- Anime.js - Lightweight
- Three.js - 3D
- Lottie-web - Vector animations
