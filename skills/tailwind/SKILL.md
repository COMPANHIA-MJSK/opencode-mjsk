# Tailwind CSS

Expert in utility-first CSS with Tailwind framework.

## Core Concepts
- Utility classes, responsive prefixes, dark mode
- Custom configuration (tailwind.config.js)
- JIT (Just-In-Time) engine
- Design tokens and theming

## Features
- **Layout**: Flexbox, Grid, Container, Columns
- **Typography**: Font size, family, weight, leading, tracking
- **Spacing**: Padding, margin, gap, space between
- **Colors**: Custom palette, opacity, gradients
- **Responsive**: sm, md, lg, xl, 2xl breakpoints
- **States**: hover, focus, active, disabled, group, peer
- **Animations**: transition, transform, keyframes
- **Dark Mode**: class-based, media-based

## Advanced
- Custom plugins
- @apply directive (use sparingly)
- Arbitrary values ([...])
- Container queries
- CSS @layer integration
- Design system integration with CSS variables

## Integration
- Tailwind + React/Next.js
- Tailwind + Vue/Nuxt
- shadcn/ui (built on Tailwind)
- Headless UI components
- Tailwind CSS IntelliSense
